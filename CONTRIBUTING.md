# Contributing

This is my personal website, so it isn't really open to feature contributions. That said, if you spot a typo, a broken link, or something clearly wrong, feel free to open an issue and let me know — I appreciate it.

## Want to build your own site like this?

This site is built on the open-source **TermHub** template. If you'd like to use it for your own portfolio, or contribute improvements to the template itself, head over to the original project:

**https://github.com/H-Freax/TermHub**

That's the right place for feature requests, bug fixes, and pull requests to the template.
